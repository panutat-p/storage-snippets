# One to many

* one company can have many fruits
* each fruit can only be associated with one company

```sql
CREATE TABLE companies (
    id SERIAL PRIMARY KEY,
    name VARCHAR(255)
);

CREATE TABLE fruits (
    id SERIAL PRIMARY KEY,
    name VARCHAR(255),
    price DECIMAL,
    company_id INT,
    FOREIGN KEY (company_id) REFERENCES companies(id)
);
```
