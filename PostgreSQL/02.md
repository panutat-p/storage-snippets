# One to one

## fruits & tags

* `tags` is parent table
* `fruits` is child table

```sql
CREATE TABLE tags (
    id SERIAL PRIMARY KEY,
    name VARCHAR(255)
);

CREATE TABLE fruits (
    id SERIAL PRIMARY KEY,
    name VARCHAR(255),
    price DECIMAL,
    tag_id INT,
    FOREIGN KEY (tag_id) REFERENCES tags(id)
);
```
