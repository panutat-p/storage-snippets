# Collection

## Schema validation

```js
db.createCollection("fruits", {
   validator: {
      $jsonSchema: {
         bsonType: "object",
         required: [ "id", "name", "color", "price", "tag", "created_at", "updated_at" ],
         properties: {
            id: {
               bsonType: "int"
            },
            name: {
               bsonType: "string"
            },
            color: {
               bsonType: "string"
            },
            price: {
               bsonType: "decimal"
            },
            tag: {
               bsonType: "array",
               items: {
                  bsonType: "string"
               }
            },
            updated_at: {
               bsonType: "date"
            },
            created_at: {
               bsonType: "date"
            }
         }
      }
   }
})
```
