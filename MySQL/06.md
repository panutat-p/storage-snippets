# Size

>  MySQL
> ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_bin
> 
> Max row size = 16KB

```sql
SELECT (CHAR_LENGTH(id) + CHAR_LENGTH(name)) as row_size
FROM person;
```

## Storage

* InnoDB storage engine
* utf8mb4 character set

| Type       | Size |
| ---------- | ---- |
| INT        | 4B   |
| BINARY(16) | 16B  |
| CHAR(27)   | 108B |
| CHAR(36)   | 144B |
